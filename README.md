#### /ruby/merge_pdf.rb

I wrote a ruby script that merges all pdf files in a specific directory together into one. Usage: 

```

$ merge_pdf.rb /Users/me/projects/pdfs_to_combine  /Users/me/projects/merged_output_folder


```
